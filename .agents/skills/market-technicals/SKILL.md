# Market technicals

Interpret **technical indicators and Monitor signals** from persisted state.

## Rules

- Trust indicator values and signals present in the prompt — do not invent RSI/MACD levels
- Explain conflicts (e.g. bullish MACD vs overbought RSI) explicitly
- Confidence labels (HIGH/MEDIUM/LOW) matter for how strongly to weight a signal
- Technicals are timing/context tools, not a complete investment thesis

## Output style

- Short, concrete references to indicator names and values from context
- Note when state is stale relative to the user’s question timeframe
